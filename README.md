HOW TO RUN

- download the zip or clone this repository
- open cmd and change to a directory you like
- clone the repository in that directory
    - type 'git clone <repository-link>
- go to the project directory
    - type 'cd ADETFinalProject'
- open the project in VS Code
    - type 'code .'
 
- open a terminal and install dependencies
    - type 'flutter pub get'
- enable Developer Mode 
    - type 'start ms-settings:developers'
- run the app
    - type 'flutter run' or 'flutter run -d chrome' if you want to run it using Chrome
